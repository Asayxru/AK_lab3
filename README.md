Лабораторна робота №3
Розгортання програмного середовища та збирання ПЗ для ARM (BeagleBone Black)
Мета роботи

Отримати навички збирання U-Boot, ядра Linux та мінімальної файлової системи BusyBox для архітектури ARM, а також виконати запуск побудованої системи в емуляторі QEMU.

Зміст репозиторію
конфігураційні файли:

u-boot.config

linux.config

busybox.config
rcS — стартовий скрипт rootfs.
log.txt - лог файл роботи системи.

Короткий опис виконаних етапів
Було встановлено ARM-кроскомпілятор, завантажено та сконфігуровано вихідні коди U-Boot, Linux kernel та BusyBox. На основі BusyBox сформовано rootfs та створено initramfs (rootfs.cpio.gz). Після цього виконано успішне завантаження ядра Linux у QEMU та отримано робочу консоль BusyBox.

Висновок

У ході роботи зібрано мінімальну ARM-систему та перевірено її запуск у QEMU. Завдання виконано повністю, опрацьовано процеси кроскомпіляції, налаштування ядра та формування initramfs.
